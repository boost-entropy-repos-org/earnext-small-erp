# small-erp
> 练手的小型后端项目

接口地址：http://localhost:10000/doc.html

### 框架
*Spring Boot 2.3.0.RELEASE*    
*Mybatis*    
*Spring Security*    
*Activiti6*    
*Redis*    
*PageHelper*    
*Hikari*    
*MySQL*    
*knife4j*    
*lombok*    
*RestFul风格*    

### 说明
- 创建时：`id`,`创建时间`,`创建人`系统实现`Mybatis`插件自动插入。
- 修改时：`修改时间`,`修改人`系统实现`Mybatis`插件自动插入。
